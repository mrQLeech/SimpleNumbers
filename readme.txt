Результат выполнения тестового задания. 

Условия тестового задания: 

Из ряда простых чисел взять 10 млн. значений и найти их сумму. 
Ограничения: 
	Запрещены массивы с числом элементов больше 10 млн.
	Время выполнения вычислений не более 60 секунд. 
	Программа должна работать на “чистой” Windows (в том виде, в котором она находится сразу после установки, без дополнительно установленного ПО)
Дополнительное условие: 
	Использовать версии Windows версии не ранее “Chicago” (кодовое наименование Windows 95).


2. Результат выполнения.

Выбран язык программирования C++, так как .Net Framework предустановлен не на всех версиях Windows. 

Результаты проверки созданной программы (файл main.cpp): 

	Проверена на Windows XP, Windows 7, Windows 10 (на более ранних версиях проверка не проводилась из-за отсутствия соответствующих виртуальных машин и дистрибутивов). 
	Время выполнения: 
	CP Intel Core i5 2-е поколение: 80 - 90 c;
	CP Intel Core i5 6-е поколение: 40 - 50 с;

3. Порядок работы: 

1. Запустить приложение SimpleNumbers.exe. На экране отобразится консоль с надписью "start calculation". Это будет означать, что расчет начался.
2. По завершению работы программы, в консоли отобразится сумма полученная при расчете и время в секундах затраченное непосредственно на расчет.
